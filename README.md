# PyMarkdownDiaryHelper

### 介绍

创建每天的日记文件，并且可以将每月或每年的合并，并添加封面和目录。

### 生成的文件结构

示例

2020年
├── 1月
├── 2月
├── 3月
├── 4月
├── 5月
├── 6月
├── 7月
├── 8月
├── 9月
├── 10月
├── 11月
└── 12月
          ├── 1.md
          ├── 2.md
          └── 3.md

### 日记头信息

## xxxx年xx月x日	星期x
> 天气	:	
> 班休	:	
> 时间	:	xx:xx

***
